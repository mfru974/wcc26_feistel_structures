# Experimental Code – 3-Round Feistel Sboxes

This repository contains the SageMath scripts used to verify the experimental results of the paper:

*Weakness of Sboxes based on 3-round Feistel Structures* (WCC 2026 submission).

---

## Requirements

- SageMath (≥ 9.x)

The code depends on **sboxUv2**.

---

## Installation

Install sboxUv2 from https://github.com/lpp-crypto/sboxU/tree/v2 following the Readme instructions.

## Usage

The experimental scripts are located in the sage_scripts/ directory and can be executed with Sage, e.g: 

sage sage_scripts/commutative.py
